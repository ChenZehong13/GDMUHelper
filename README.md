# 广医小助手

旨在方便广医人日常学习交流的微信小程序

---

## 项目介绍：

一个基于 微信小程序 、express (后端) 的工具类小程序。

## 项目优势：

- 好看的页面和交互效果(大雾

- 每日实时更新的精选文章阅读
- 比官方更早的获知考试成绩（当学校外网链接没崩时）
